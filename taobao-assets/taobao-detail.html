📥 安装指南

⚠️ 首次打开须知： 由于 App 未经过 Apple 官方公证，首次打开时请 右键（或 Control + 点击）App → 选择「打开」 → 在弹出的对话框中再次点击「打开」即可。此后正常双击打开即可。

🔑 如何获取 API Key

  1. 打开 platform.deepseek.com 并注册/登录
  2. 点击左侧菜单「API Keys」
  3. 点击「创建新的 API Key」
  4. 复制以 sk- 开头的一串字符
  5. 粘贴到 DeepSeek Monitor 的设置窗口中，保存即可

❓ 常见问题

Q: 支持 Windows 吗？
A: 不支持。DeepSeek Monitor 是 macOS 独占应用，仅支持 macOS 13 及以上系统。
Q: 我的 API Key 安全吗？
A: 绝对安全。Key 仅保存在你 Mac 本机的 UserDefaults 中，不会上传到任何第三方服务器。所有 API 请求直接从你的电脑发到 DeepSeek 官方服务器 (api.deepseek.com)。
Q: 会影响电脑性能吗？
A: 完全不会。纯 SwiftUI 原生开发，每 5 分钟仅发送一次极小的 HTTP 请求（几百字节），空闲时 CPU 和内存占用几乎为零。
Q: 能否查看历史用量？
A: 当前版本显示「本次会话消耗」（App 启动至今的累计消耗）。每日/每月历史统计功能计划在后续版本中加入。
Q: 支持 Intel Mac 吗？
A: 支持！同时兼容 Apple Silicon（M1/M2/M3/M4/M5）和 Intel 芯片的 Mac。
