## 七、客服话术（快捷回复）

**Q: 支持 Windows 吗？**
A: 抱歉，仅支持 macOS 13 及以上系统。Windows 用户请勿下单。

**Q: 怎么获取 API Key？**
A: 访问 platform.deepseek.com → 注册/登录 → API Keys → 创建 Key → 复制 sk- 开头的一串字符即可。

**Q: API Key 安全吗？**
A: 绝对安全。Key 只保存在你本机的 UserDefaults 中，不会上传到任何服务器。所有 API 请求直接从你的 Mac 发到 DeepSeek 官方服务器。

**Q: 影响电脑性能吗？**
A: 完全不。纯 SwiftUI 原生开发，后台每 5 分钟发一次极小的 HTTP 请求，CPU 和内存占用几乎为零。

**Q: 能不能显示每日/每月用量？**
A: 当前版本显示「本次会话消耗」（从 App 启动到现在的累计消耗）。每日/每月统计计划在后续版本加入，购买高级版可免费升级。
